## How It Works (1 minute)

"The system has five agents that work together:
1. **Planner** - Breaks your query into research topics and search queries
2. **Web Search** - Finds relevant articles and extracts content
3. **Analyzer** - Extracts key insights from the content
4. **Summarizer** - Creates a concise summary
5. **Report Generator** - Produces the final structured report

They're orchestrated using LangGraph, which manages the workflow and handles retries and errors."
